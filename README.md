Source code for my personal portfolio website.

## Overview
This repository hosts the static files for my personal portfolio, showcasing my work in cybersecurity, defensive strategies, and secure system development.

## Content Highlights
The site details the following experience and projects:
* **Objective:** Focus on secure systems, defensive strategies, and collaborative system hardening.
* **Skills:** SIEM Implementation, Log Analysis, Network Traffic Monitoring, and Enterprise Penetration Testing.
* **Projects:**
    * **Threat Detection Lab:** Environment for detection, monitoring, and defensive techniques (Wazuh, Suricata, Sysmon).
    * **Corporate Attack Lab:** Enterprise CTF simulation replicating multi-stage attacks.
    * **Python Credential Manager:** Secure command-line tool for API key and secret management.

## Tech Stack
* **HTML/CSS**
* **GitHub Pages** (Hosting)

## Installation & Usage
No installation required. This is a static site served directly by GitHub.

To view locally:
1.  Clone the repository:
    ```bash
    git clone [https://github.com/baileym12/baileym12.github.io.git](https://github.com/baileym12/baileym12.github.io.git)
    ```
2.  Open `index.html` in your browser.
